Chapitre 3 Feuilles perforées

Évaluer si le nombre de feuilles ayant subi des attaques d’insectes, de type perforation, diffèrent significativement selon la méthode de contrôle appliquée.

Jeu de données kam.csv qui contient différentes mesures dont les nombres de feuilles perforées, comptés en 4 séances.

On pourrait également évaluer si les différentes méthodes ont des intensités d’attaques de ce type, significativement différentes avec le temps. On comparera les effets des méthodes séance par séance, puis à l’aide d’une figure on appréciera s’il y a une évolution en fonction du temps.

3.1 Les données

df <- read_csv("data/kam.csv")
df <- df %>% 
  mutate(id = rep(1:8, 4), .before = 1, seance = factor(seance),
         methode = factor(methode, levels = c("Temoin", "Filet-anti-insectes", "Lutte-mecanique", "Cigogne-50-EC")), 
         bloc = factor(bloc)) %>% 
  select(id, seance, bloc, methode, nfp) %>% 
  mutate(id = factor(id))

Le tableau est déjà structuré en format long en Excel. J’ai ajouté un identifiant (id) pour les échantillons des seances.

3.4 L’ANOVA

3.4.1 Séance 1 (1ère semaine de mesure)

3.4.1.1 Le modèle

df1 <- df %>% filter(seance == "Semaine 1")

lm1 <- glm(nfp ~ methode, data = df1, family = poisson)
Anova(lm1)
## Analysis of Deviance Table (Type II tests)
## 
## Response: nfp
##         LR Chisq Df Pr(>Chisq)   
## methode   11.667  3   0.008617 **
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

La p-value < 0.05 => différence très significative entre les effets des méthodes de contrôle sur le nombre de feuilles perforées à la Semaine 1.
